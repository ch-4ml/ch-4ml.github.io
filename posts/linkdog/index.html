반려견 웨딩 서비스 - 이어줄개

프로젝트 팀 :

  • 박찬형(ch-4ml)
  • 최한솔(9992)
  • 원소희(infiduk)

프로젝트 기간

2019.08 ~ 2019.09

프로젝트 개요

반려견 결혼 프로젝트

현황

  • 펫팸족 1000만 시대
  • 1인 크리에이터 시대

제안

반려동물의 웨딩(결혼) 서비스를 개발하고자 합니다. 서비스 내에는 웨딩, 교배, 출산, 산후조리, 분양, 반려동물 등록 지원, 신혼여행 등의 절차가 포함되고, 각 절차에는 필요한 경우 블록체인 기술을 이용해 정보를 기록함으로써 사용자에게 신뢰도 높은 정보를 제공할 수 있습니다.

기획 / 설계 문서 :

Go to notion

개발 언어 / 구성 환경

Javascript, Solidity / Luniverse, Node.js, React-Native, MySQL 5.7

구현 내용 :

  • 반려견 회원 가입
  • 청혼
  • 결혼 패키지 예약
  • 결혼 정보 저장

회원가입

로그인

메인

프로필

프로젝트 내 역할 :

  • Luniverse API
  • Node.js Server
